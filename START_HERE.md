# 🚀 从这里开始 - FoxAI 快速指南

## 👋 欢迎使用 FoxAI 影视聚合平台！

这是一个基于 MoonTVPlus 定制的影视聚合平台，具有独特的 FoxAI 品牌风格。

---

## ⚡ 3 步快速启动

### 1️⃣ 安装依赖
```bash
cd /Users/foxai/Desktop/MoonTVPlus
pnpm install
```

### 2️⃣ 生成配置
```bash
pnpm gen:manifest
```

### 3️⃣ 启动服务
```bash
pnpm dev
```

### ✅ 访问应用
打开浏览器访问: **http://localhost:3000**

---

## 🎯 或者使用一键脚本
